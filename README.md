# 📊 Analyse des performances du CAC 40 (2017–2019)

## 📌 Problématique

Comment les performances du CAC 40 ont-elles évolué de 2017 à 2019, et quelles tendances ou facteurs peuvent être identifiés à partir de cette analyse ?

## 🎯 Objectif

Analyser les performances du CAC 40 à travers des visualisations interactives afin d’identifier les tendances et les facteurs influençant le marché.


## 📈 Analyse

* Étude de l’évolution du CAC 40 sur la période 2017–2019
* Identification des tendances haussières et baissières
* Analyse des indicateurs clés de performance (KPI)
* Mise en évidence des périodes de volatilité

## 🛠️ Outils utilisés

* Power BI
* DAX
* Data Visualisation

## 📎 Fichier

Le fichier Power BI (.pbix) est disponible dans ce repository.

## 🚀 Résultats clés

* Identification des phases de croissance et de ralentissement
* Visualisation claire des performances du marché
* Support à la prise de décision pour les investisseurs
